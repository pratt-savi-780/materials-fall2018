In-class Exercise: Data

Part 1: Convert shapefiles and KMZs to GeoJSON

  1. Download this week's data and unzip it.
  2. Using QGIS or an online converter like mapshaper, convert
    • Hurricane_Evacuation_Routes.kmz
    • lisseddata
    to GeoJSONs.

Part 2: Download and convert JSON data

  1. Install OpenRefine on your computer and open it.
  2. Go to HabitatMap.
  3. Open developer tools and go to the network tab.
  4. Under Featured Maps in the bottom left of the site, click on Creek Speak: Voices from Newtown Creek
  5. When you do, you should see requests made under the network tab. Find markers.json, click on it, and go to the Response tab. Select and copy all of the JSON response.
  6. Paste the response into OpenRefine, under Clipboard, select the first full JSON element—the first entry you expect to be one row, and click Create Project.
  7. Rename the latitude and longitude columns to the full names of "latitude" and "longitude".
  8. Download as CSV.
  9. Open the CSV in QGIS using Layer > Data Source Manager.
  10. Export the layer as a GeoJSON.

Part 3: Filtering in QGIS

  1. Open Bicycle Routes in QGIS.
  2. Filter the file to only features where facilitycl is Sharrows.
  3. Export as a GeoJSON.

Part 4: Joining data

  1. Open ACS_16_5YR_B25003/ACS_16_5YR_B25003_edit.xlsx and nys_census_tracts in QGIS.
  2. Open the layer properties for nys_census_tracts.
  3. Go to the Joins tab.
  4. Add a new join using the + button.
  5. Select the spreadsheet for the Join layer.
  6. Select GEO.id for the Join field.
  7. Select GEO_ID for the Target field.
  8. Press OK and confirm that the join was completed.

Part 5: Look for data related to your project

  1. Explore data related to your project using data.gov, NYC Open Data, and Google Dataset search.
  2. What kind of work might need to happen for the data to be used in Leaflet?
  3. Is this work you can do with QGIS?
