In-class Exercises

Part 1: Creating a simple map with geojson.io

  1. If you don't have one already, create a GitHub account.
  2. Go to geojson.io and log in with your GitHub account.
  3. Draw some points around your home.
  4. Add some data to those points, such as the names of the places they represent.
  5. Under the Save menu, click Gist and view your map in the "gist" it creates. Keep this open in a tab for later.

Part 2: HTML

  1. Use your GitHub account to log in to Glitch.
  2. Click Start a New Project then Create a Website.
  3. Open the file index.html in Glitch.
  4. Add a link in the body tags: 
    <a href="http://example.com">visit example.com</a>
    Change the URL and text to something more interesting. Refresh the page in your browser.
  5. Go back to your Gist from Part 1 and copy the embed code and paste it into your HTML.
  6. Experiment with other tags that we talked about and from the HTML reference.

Part 3: Putting images online with Glitch

  1. Continue with the Glitch project from Part 2.
  2. Click the assets folder in the left of the page.
  3. Upload an image and get the URL for it.
  4. Add the image to your HTML page.
