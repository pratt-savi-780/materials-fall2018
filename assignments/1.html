Assignment 1: HTML & CSS

Due September 4 at 6pm

We talked about HTML and CSS this week, and we'll exercise some of our new knowledge around them by using HTML and CSS to make our own pages.

Resources

  • My HTML Reference should get you started.
  • Codecademy has a few courses: Learn HTML (Just do the first section, "Elements and Structure", don't bother with the pro lessons) and Learn CSS (The first five sections, don't bother with the pro lessons).
  • MarkSheet has a good free HTML and CSS tutorial.
  • CSS-Tricks has good beginner-to-intermediate tutorials, and you might appreciate HTML & CSS -- The VERY Basics.
  • The first three sections on this HTML Dog page list the more common CSS properties you'll be using.
  • This CSS cheat sheet is a terse but you might find it useful.
  • There are more exhaustive references on the Mozilla Developer Network.
  • If you're looking for a resource on HTML and CSS that you can read through, I'd recommend the first six chapters of HTML & CSS is Hard. For now I would stick with editing code in Glitch, so you can skip the parts that mention a text editor named Atom.

Create a page with HTML and CSS

Let's create an webpage from scratch using HTML and CSS. Pick something you're interested in and make a one-page site in Glitch about that topic.

  1. Create a site in Glitch.
  2. Edit the HTML page (index.html):
    • add some text
    • add at least one image (upload it to Glitch first)
    • add at least one link
    • experiment with some other HTML tags as outlined in the HTML Reference
  3. Add CSS to style it. Consider fonts, colors, and positions when making your changes. Use at least three selectors, and select based on your elements' classes for at least one. The styles should be in a CSS file linked in the <head> of your document, like this: 
    <!doctype html>
<html>
    <head>
        <link rel="stylesheet" href="/styles.css">
    </head>
    <body>
    </body>
</html>
  4. Submit a link to the public page on Glitch (not the link you use for editing).
