Final Project

Due October 16 at 6pm

Create an online interactive map

Throughout this course we have covered the essential technologies and skills needed to create an online interactive map. Use this project to make a map of your choosing that demonstrates your knowledge of these technologies and skills.

Your map can be on any topic that you like and can include any datasets that you find suitable for your map.

Your map must:

  • Use HTML, CSS, and JavaScript that you have written. If you are using an HTML and CSS template for the structure of the page, you must customize the HTML and CSS in some significant way.
  • Include at least one data source and credit each data source.
  • Include a base map that you create with Mapbox.
  • Include JavaScript event listeners for interactivity.
  • Include a short description of the project and the methodology behind it.
  • Be hosted on GitHub and use GitHub Pages.

Outside of the above, you can do the project on whatever you find interesting or useful.

Submitting your work
