Create a page with a Leaflet map on it

Let's create an webpage from scratch using HTML and CSS, then add a Leaflet map to it using JavaScript.

  1. Explore NYC's Open Data portal and find at least one dataset you'd like to add to a map. The dataset must have map data in it for this to work—you can filter the datasets to just maps to make this search easier.
  2. Create a site in Glitch.
  3. Edit the HTML page and style it with CSS.
  4. Load Leaflet and add a map to the page. Use the example Glitch site we have been working with to help.
  5. Add the data from the dataset you picked on the NYC Open Data portal to the map. On the dataset's page, you can find the URL to use by going to Export > SODA API > API Endpoint. Don't forget to change .json to .geojson.
  6. Submit a link to the public page on Glitch (not the link you use for editing, the one you get when you click Show Live).
