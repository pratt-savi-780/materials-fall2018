In-class Exercise: Data, More JavaScript

Part 1: Using leaflet-control-geocoder

leaflet-control-geocoder is a plugin that adds a search bar to a Leaflet map where users can search for an address to zoom into. It doesn't deal with your map's data. Let's try adding it to a map here.

  1. Open leaflet-control-geocoder.
  2. Look for an example in the documentation and open it.
  3. Open a Glitch site of your own that uses Leaflet, or start with this minimal Leaflet example.
  4. Using the documentation for leaflet-control-geocoder, add the plugin to the site in Glitch and confirm that you get a search bar as you do in the example.

Part 2: Using Leaflet.markercluster

Leaflet.markercluster is a handy plugin that can help make dense point datasets easier to view in Leaflet. Here we'll work from an example of using Leaflet.markercluster to customize it.

  1. Open Leaflet.markercluster.
  2. Look for an example in the documentation and open it to see what it does.
  3. Start from this example and change the data that is loaded and added to the markercluster. This could be data you are working with for your project or other data we have looked at in class.
  4. If you have time, look at the options for L.markerClusterGroup() and try changing some of the options.

Part 3: Explore Leaflet plugins

  1. Look at the plugin list for Leaflet.
  2. Follow a few links to plugins that sound interesting to you.
  3. For each plugin, try to find a demo or example page. Is there one? Does the plugin seem useful to you?
  4. Starting from a Glitch site of your own or this simple Leaflet site, attempt to use one of the plugins you looked at.

Part 4: Mustache templates

  1. Remix this Glitch site into your account.
  2. Look at how the Glitch site above stores the template in HTML.
  3. Change the context data and see how the rendered HTML changes.

Part 5: Mustache templates in Leaflet popups

  1. Remix this Glitch site into your account.
  2. Change the popup templates to include the status and resolution_description for the clicked feature.
  3. Add a class to an element in the template and style it.
  4. If there is time, use a different data source and customize the templates for it.
