In-class Exercise: Turf.js

Part 1: Turf, buffer

  1. Remix this Glitch site
  2. Open the documentation for turf.buffer.
  3. Change some parameters (radius, units) in turf.buffer().

Part 2: Turf, center

  1. Remix this Glitch site
  2. Use turf.center to find the center of the collisions and put that on the map instead of the collision points.

Part 3: Experiment with Turf

  1. Start from a Glitch site of your own or one of these:
    • Leaflet with a little bit of data
    • counting points in polygons with Turf and the Open Data API
  2. Experiment with Turf. Use a few functions and see what they do. Some are very specific and less obviously useful, while others are more standard GIS-type functions. Try go get some data to show up on the map that your create or modify with Turf. If you're having a hard time finding functions to use, try one of these:
    • concave
    • convex
    • randomPoint
    • isolines
    • hexGrid
