In-class Exercise: Mapbox

Part 1: Creating styles with Mapbox

  1. Create an account with Mapbox if you don't already have one.
  2. Go to Styles and create a new style. When picking a template style, start with Light for this activity.
  3. Edit the style. Start with the parks layer and change the color and stroke of parks. Zoom into a city to see your handywork.
  4. Make the colors of the parks layer change by zoom level.
  5. Next look at streets (Roads > road-street). What is the relationship between road-street and road-street-case?
  6. Experiment with other types of data and properties in your map style.

Part 2: Using Mapbox styles in Leaflet

  1. Publish your style from Part 1.
  2. Click the Use tab once you publish it, then Third party and CARTO.
  3. Copy the URL under Integration URL.
  4. Open a Leaflet map in Glitch. If you need one to start with, remix this one.
  5. In your JavaScript file, find the line with L.tileLayer() on it and put the URL from Mapbox in the function instead of the URL that is currently there.
